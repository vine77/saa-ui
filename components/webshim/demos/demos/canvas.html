Demo/About/Documentation HTML5 canvas

Webshim Lib for the feature canvas implements one simple jQuery-method .getContext().

var ctx = $('#my-canvas').getContext('2d'); ctx.fillStyle = "rgb(200,0,0)"; ctx.fillRect (10, 10, 55, 50); ctx.fillStyle = "rgba(0, 0, 200, 0.5)"; ctx.fillRect (30, 30, 55, 50);

Webshim Lib can implement excanvas or FlashCanvas (default)/FlashCanvas Pro:

//always run configuration before calling $.webshims.polyfill(); window.FlashCanvasOptions = { disableContextMenu: true }; $.webshims.setOptions('canvas', { type: 'flashpro' //excanvas | flash | flashpro });

Options for the canvas feature

  • type: "flash" | "flashpro" | "excanvas"
